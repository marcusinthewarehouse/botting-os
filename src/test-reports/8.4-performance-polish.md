## 6. N+1 Query Patterns

**Status: WARN - 1 issue found**

`src/lib/db/repositories/inventory.ts:28-32`:
```typescript
export async function bulkRemove(ids: number[]): Promise<void> {
  for (const id of ids) {
    await db.delete(inventoryItems).where(eq(inventoryItems.id, id));
  }
}
```
This executes N separate DELETE queries. Should use `inArray()` for a single query.

**Recommendation**: Replace with:
```typescript
import { inArray } from 'drizzle-orm';
await db.delete(inventoryItems).where(inArray(inventoryItems.id, ids));
```
